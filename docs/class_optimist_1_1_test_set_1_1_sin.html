Detailed Description

template<typename Scalar>
requires TypeTrait<Scalar>::IsScalar
class Optimist::TestSet::Sin< Scalar >

Class container for the sine function, which is defined as:

\[f(x) = \sin(x) \text{.} \]

The function has roots at \(x = \pi/2 + i\pi\), with \(f(x) = 0\), and minima at \(x = 3\pi/2 + 2i\pi\), with \(f(x) = -1\) and \(i = 0, 1, \ldots, n\). The initial guesses are generated on the range \(x \in \left[-\pi, \pi\right]\).
